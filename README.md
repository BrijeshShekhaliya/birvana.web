Birvana is a premium, beautifully designed music streaming application built for audiophiles. It combines a stunning Material You-inspired user interface with a powerful custom audio playback engine to deliver millions of songs in pristine, lossless quality.

This repository contains the source code for the official marketing and developer website for Birvana.

## ✨ Application Features

- **High-Fidelity Audio:** Stream pristine, lossless FLAC and high-bitrate MP3s directly to your device.
- **Intelligent Discovery:** Smart recommendation algorithms that understand your listening habits.
- **Unified Search Pipeline:** Lightning-fast aggregate search across multiple massive media catalogs.
- **Offline Library:** Robust caching and local storage engine for seamless playback without an internet connection.
- **Audiophile Engine:** Custom low-latency playback engine with granular EQ controls.
- **Immersive UI:** A gorgeous, fluid, and dynamic user interface with glassmorphism and real-time palette extraction.

## 💻 Website Tech Stack

The Birvana marketing website is built with a modern, high-performance web stack:

- **Framework:** React 18 & Vite
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion (Scroll-driven dynamic scenes)
- **Icons:** Lucide React
- **Typography:** Inter & JetBrains Mono

## 🚀 Local Development

To run this website locally on your machine:

```bash
# Clone the repository
git clone https://github.com/BrijeshShekhaliya/birvana.web.git

# Navigate into the directory
cd birvana.web

# Install dependencies
npm install

# Start the development server
npm run dev
```
